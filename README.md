# Little Dino Quest

Little Dino Quest is a small 2D platformer demo built with Unity.
The project was created as a learning exercise to explore how to build 2D games in Unity, including character movement, level design, enemies, and gameplay mechanics.

The main objective of the game is to guide a small dinosaur through different levels while avoiding traps, defeating enemies, and collecting coins.

## Gameplay
In Little Dino Quest, the player controls a dinosaur that must traverse platforming levels filled with challenges.

The goal is to reach the end of each level while surviving the different hazards.

## Game Features

The demo also includes several basic systems commonly used in games:

- Main Menu
- Pause Menu
- Settings Menu
- Sound Effects system *(not included in this repository due to asset licensing)*
- Coin counter and lives system
- Checkpoints
- Basic enemy AI


## Project Structure
```text
Assets/
 ├── Scripts/        # Player, enemies, and gameplay logic
 ├── Scenes/         # Game levels
 ├── Prefabs/        # Reusable game objects
 ├── Sprites/        # 2D graphics and animations
 └── Resources/      # Texts, Fonts that can be used
```
